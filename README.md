# Orbital-Simulation-of-two-body-system
This simulation shows two body system (planet and sun) based on simple Newtonian Mechanics (Newton's Law of Gravitation is used here), and Euler Integration. The resulting 2D simulation shows the elliptical orbit of planet that gyrates because of gravitataional influence.
Tools used are:
Python, 
Matplotlib, and
NumPy
